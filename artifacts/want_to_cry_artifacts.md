# Artifacts - `want_to_cry`

This repository focuses on artifacts that contain the string `want_to_cry`.

## Included artifact patterns
- File names **ending with** `.want_to_cry`
- File names **containing** `want_to_cry` anywhere (for example, `Install.ps1.want_to_cry`)
- Any folder paths or registry data containing `want_to_cry` (if present)

## How to list artifacts
Run:
- `kql_queries/01_find_want_to_cry_files.kql`

Then export results to CSV for your evidence pack (redact as needed).
